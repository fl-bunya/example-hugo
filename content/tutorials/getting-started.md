## Hugoとは

Hugoは、Go言語で作られた高速な静的サイトジェネレーターです。

## インストール方法

### macOS (Homebrew)

```bash
brew install hugo
```

### Windows (Chocolatey)

```bash
choco install hugo-extended
```

## 最初のサイトを作成

```bash
hugo new site mysite
cd mysite
```
